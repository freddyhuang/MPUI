# mp-fast-ui 组件库

### 快速开始

#### 1.安装组件库


```bash
 npm i mp-fast-ui
```

#### 2.引进组件库
```javascript
// 全部引进
import 'mp-fast-ui/dist/css/index.css'
import MPUI from 'mp-fast-ui';
Vue.use(MPUI)

// 按需引进
import 'mp-fast-ui/dist/css/demo.css';
import { Demo } from 'mp-fast-ui';
Vue.use(Demo);
```
